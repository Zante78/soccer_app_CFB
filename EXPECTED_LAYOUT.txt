ON PAGE LOAD:
  1. Show skeleton loaders (3 pulsing cards)
  2. Fetch RPA traces from server
  3. Generate signed URLs for screenshots
  4. Render error cards
  5. Auto-refresh every 30 seconds

CLICKING "NEUE BASELINE AKZEPTIEREN":
  1. Button text → "Wird akzeptiert..."
  2. Button becomes disabled
  3. Server Action: acceptNewBaseline(traceId)
  4. Update trace status to SUCCESS
  5. Toast appears: "Baseline akzeptiert"
  6. Card disappears from list
  7. If no more errors, show success state

CLICKING "BOT ERNEUT AUSFÜHREN":
  1. Button text → "Startet..."
  2. Button becomes disabled
  3. Server Action: retryBotExecution(registrationId)
  4. Update registration status to READY_FOR_BOT
  5. Toast appears: "Bot wird erneut ausgeführt"
  6. Page refreshes to show updated state

DRAGGING COMPARISON SLIDER:
  1. Divider follows mouse/touch
  2. Left side shows more/less of baseline
  3. Right side shows more/less of actual
  4. Smooth transition
